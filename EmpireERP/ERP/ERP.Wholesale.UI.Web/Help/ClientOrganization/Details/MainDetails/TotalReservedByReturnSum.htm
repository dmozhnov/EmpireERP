В данном поле отображаются два расчетных параметра, разделенные знаком «||». Первый отображает сумму принятых на склад возвратов товаров по всем накладным реализации. Второй – общую сумму всех оформленных возвратов товаров от данной организации клиента.

Суммы рассчитываются в отпускных ценах реализации возвращаемых товаров.
