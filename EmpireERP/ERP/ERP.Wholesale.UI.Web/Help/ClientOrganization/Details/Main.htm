На этой странице представлена следующая информация по выбранной организации клиента:

  • Основные детали организации клиента – отображается информация, введенная при создании (редактировании) организации клиента, а также важные расчетные показатели по организации клиента.
  • Таблица «Реализации организации по сделкам» – отображается список продаж данной организации по всем сделкам.
  • Таблица «Договоры по сделкам» – отображается список всех договоров.
  • Таблица «Оплаты организации по сделкам» – отображается список всех оплат, принятых от данной организации по всем сделкам.
  • Таблица «Корректировки сальдо по сделкам» – отображается список всех дебетовых и кредитовых корректировок общего сальдо организации клиента.
  • Таблица «Счета в российских банках» – отображается список счетов организации клиента в российских банках.
  • Таблица «Счета в иностранных банках» – отображается список счетов организации клиента в иностранных банках.

Создать новую организацию на этой странице нельзя. Чтобы добавить организацию клиента, нужно перейти на страницу «Детали клиента» и нажать кнопку «Добавить», расположенную над таблицей «Организации».

Чтобы удалить организацию клиента, нужно нажать кнопку «Удалить», находящуюся над основными деталями организации клиента. Удаление невозможно, если с организацией клиента заключен хотя бы один договор.
