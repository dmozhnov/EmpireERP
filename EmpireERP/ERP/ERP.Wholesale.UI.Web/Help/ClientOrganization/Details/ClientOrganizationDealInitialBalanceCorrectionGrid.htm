В этой таблице выводятся дебетовые и кредитовые корректировки сальдо по сделкам. Желтым цветом подсвечены строки, соответствующие кредитовым корректировкам сальдо (сумма корректировки выводится со знаком «-»).

Сразу над таблицей расположены две кнопки «Новая дебетовая корректировка» и «Новая кредитовая корректировка». По нажатию на кнопку «Новая дебетовая корректировка» будет предложено создать дебетовую корректировку сальдо. По нажатию на «Новая кредитовая корректировка» - создать кредитовую корректировку и разнести ее на накладные реализации.

Кнопки не отображаются или недоступны в случаях если:

  • недостаточно прав доступа для выполнения данных операций;
  • этап сделки не позволяет выполнить данные операции.

ВНИМАНИЕ! Изменение введенной информации невозможно. Если требуется изменить информацию по корректировке или информацию по ее разнесению, то необходимо удалить ошибочную корректировку сальдо и создать новую с правильно введенными данными.

В таблице «Корректировки сальдо по сделкам» все записи выводятся в порядке, обратном их созданию – последняя созданная запись выводится первой в таблице.
