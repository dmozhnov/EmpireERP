На этой странице представлена информация по выбранной накладной реализации товаров. Для выполнения ключевых операций над этой накладной служат кнопки и ссылки, расположенные на странице. Такими операциями являются:

  • Добавление позиций в накладную. Операция может быть выполнена одним из трех способов:

    • автоматическое добавление позиции из списка,
    • добавление позиции с точным указанием партии,
    • добавление позиции с точным указанием партии и источника.

  • Подготовка накладной к проводке. Операция подразумевает перевод накладной в стадию «Готово к проводке» и может быть выполнена только при соблюдении всех следующих условий:

    • статусу накладной соответствует «Черновик»,
    • права доступа позволяют выполнить операцию подготовки к проводке для данной накладной.

  • Проводка накладной. Операция подразумевает перевод накладной в стадию «Готово к отгрузке» или «Ожидание товаров для отгрузки» (стадия накладной будет зависеть от стадий накладных-источников) и может быть выполнена только при соблюдении всех следующих условий:

    • статусу накладной соответствует «Готово к проводке»,
    • права доступа позволяют выполнить операцию проводки для данной накладной.

  • Отгрузка накладной. Операция подразумевает перевод накладной в стадию «Отгружено», а в случае если в текущей сделке имеются неразнесенные оплаты, то эти оплаты будут автоматически разнесены на накладную при отгрузке вплоть до 100% оплаты накладной. Операция отгрузки может быть выполнена только при соблюдении всех следующих условий:

    • статусу накладной соответствует «Готово к отгрузке»,
    • права доступа позволяют выполнить операцию отгрузки данной накладной.

  • Отмена отгрузки. Операция подразумевает возврат накладной в стадию «Готово к отгрузке». Операция может быть выполнена только при соблюдении всех следующих условий:

    • статусу накладной соответствует «Отгружено»,
    • данная накладная не является источником для накладной возврата товаров в стадии «Проведена» или более,
    • права доступа позволяют выполнить операцию отмены отгрузки для данной накладной.

  • Отмена проводки. Операция подразумевает возврат накладной в стадию «Готово к проводке», а случае если на накладную разнесена оплата (или кредитовая корректировка сальдо), то разнесение будет автоматически снято. Операция отмены проводки может быть выполнена только при соблюдении всех следующих условий:

    • статусу накладной соответствует «Ожидание товаров для отгрузки» или «Готово к отгрузке»,
    • права доступа позволяют выполнить операцию отмены проводки для данной накладной.

  • Отмена подготовки к проводке. Операция подразумевает возврат накладной в статус «Черновик». Операция может быть выполнена только при соблюдении всех следующих условий:

    • статусу накладной соответствует «Готово к проводке»,
    • права доступа позволяют выполнить операцию отмены подготовки проводки для данной накладной.

  • Удаление накладной (включая все ее позиций). Операция может быть выполнена, только если накладная имеет статус «Черновик» и права доступа позволяют выполнить операцию удаления данной накладной.

  • Редактирование информации о накладной. Операция подразумевает открытие формы редактирования параметров накладной и может быть выполнена, только если накладная имеет статус «Черновик» и права доступа позволяют выполнить операцию удаления данной накладной. Для редактирования доступны следующие параметры:

    • Номер накладной
    • Адрес доставки
    • Правило округления цен в накладной до целого
    • Применяемая квота
    • Форма взаиморасчетов (если позволяет квота)
    • Применяемая ставка НДС по умолчанию
    • Команда, которая регистрирует отгрузку
