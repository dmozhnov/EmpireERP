В этой таблице выводятся все текущие позиции созданной накладной реализации товаров.

Сразу над этой таблицей расположена кнопка «Добавить», по нажатию на которой будет предложено добавить одну позицию накладной с точным указанием партии или с точным указанием и партии, и накладных-источников товара.

Кнопка не отображается или недоступна в случаях если:

  • недостаточно прав доступа для добавления позиции в данную накладную реализации товаров;
  • статус накладной реализации товаров не позволяет добавлять и редактировать позиции накладной.

В столбце «Действие» для каждой позиции отображаются ссылки:

  • «Ред.» – по нажатию будет открыта форма редактирования позиции накладной,
  • «Удал.» – по нажатию будет выполнено удаление позиции из накладной

или (если статус накладной не позволяет добавлять/редактировать позиции)

  • «Дет.» – по нажатию будет отображена форма детальной информации по данной позиции,
  • «Источ.» – по нажатию будет отображена информация о накладных-источниках для данной позиции.

В таблице «Позиции накладной» все записи выводятся единым списком в порядке их добавления в накладную.
