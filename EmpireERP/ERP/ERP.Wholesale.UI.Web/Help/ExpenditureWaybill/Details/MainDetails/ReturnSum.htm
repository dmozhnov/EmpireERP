В данном поле выводится информация о возвратах товаров, оформленных по данной накладной реализации. В поле отображаются два расчетных параметра, разделенные знаком «||». Первый отображает сумму принятых на склад возвратов товаров по данной накладной реализации. Второй – общую сумму оформленных (проведенных и принятых на склад) возвратов по данной накладной.

Суммы рассчитываются в отпускных ценах реализации возвращаемых товаров.
