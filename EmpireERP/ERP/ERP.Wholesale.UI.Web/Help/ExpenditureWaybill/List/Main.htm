Накладная реализации товаров в Bizpulse – это документ, который фиксирует отгрузку товаров клиенту по заключенному с ним договору (договор отражен в сделке). Создание накладной реализации товаров возможно только в сделках на этапе «Исполнение договора».

На данной странице отображен список всех накладных реализации товаров, которые оформлены в Bizpulse и доступны Вам для просмотра.

Каждая накладная реализации товаров имеет заданный в Bizpulse жизненный цикл, который определяется статусами накладной.
