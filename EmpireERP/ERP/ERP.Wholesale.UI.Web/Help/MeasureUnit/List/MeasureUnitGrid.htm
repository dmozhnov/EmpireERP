В этой таблице выводятся все единицы измерения, внесенные в Bizpulse.

Сразу над этой таблицей расположена кнопка «Новая ЕИ» (она отображается и доступна в том случае, если Ваши права доступа позволяют создавать единицы измерения).

Для редактирования единицы измерения нужно нажать ссылку «Ред.» в столбце «Действие».

Для удаления единицы измерения нужно нажать ссылку «Удал.» в столбце «Действие». Удаление единицы измерения возможно только в случае, если она не задана в качестве единицы измерения ни для одного товара из справочника «Товары».
