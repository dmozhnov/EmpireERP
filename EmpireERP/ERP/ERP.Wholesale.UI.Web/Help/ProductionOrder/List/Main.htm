Заказ на производство в Bizpulse – это сущность, которая включает в себя всю информацию, связанную с заказным производством продукции. Один заказ отражает один контракт с производителем для производства оговоренной партии готовой продукции (в одну или несколько партий поставки).

С помощью заказов ведется учет производства продукции. По каждой партии в заказе можно автоматически создать накладную прихода продукции на места хранения и рассчитать себестоимость этой продукции на основе понесенных затрат.
