В поле «Текущая стоимость» отображается суммарная стоимость заказа по текущим затратам (стоимость производства по всем партиям, суммарные затраты по листам транспортных, таможенных и дополнительных расходов). Первое значение соответствует сумме в валюте, второе – сумме в рублях по учетному курсу заказа.
