В данной таблице отображаются все платежи, внесенные в план, по данному заказу.

Сразу над этой таблицей расположена кнопка «Добавить платеж в план», по нажатию на которой будет предложено добавить платеж в план совершения платежей по данному заказу.

Для редактирования платежа нужно нажать ссылку «Ред.» в столбце «Действие».

Для удаления платежа из плана нужно нажать ссылку «Удал.» в столбце «Действие».
