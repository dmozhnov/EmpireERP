Данное поле предназначено для одного из имеющихся в Bizpulse мест хранения. В дальнейшем, при формировании приходной накладной по заказу, данное место хранения будет указано в качестве приемщика товаров. Также место хранения определяет учетные цены на производимые товары при расчете суммы производства в учетных ценах и потенциальной прибыли.
