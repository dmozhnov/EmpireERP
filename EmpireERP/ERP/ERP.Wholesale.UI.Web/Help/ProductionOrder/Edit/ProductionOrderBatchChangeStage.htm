Данная форма предназначена для изменения этапа заказа.

Текущий этап отражает этап, на котором находится партия товара в данный момент.

Следующий этап отражает этап, который может быть пройден накладной на следующем шаге своего жизненного цикла. Чтобы перейти на следующий этап, нужно нажать кнопку «Перейти на следующий».

Предыдущий этап отражает этап, который был пройден накладной на предыдущем шаге своего жизненного цикла. Чтобы вернуться на предыдущий этап, нужно нажать кнопку «Вернуть предыдущий».
