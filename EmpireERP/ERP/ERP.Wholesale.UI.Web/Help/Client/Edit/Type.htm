В данном поле необходимо указать тип клиента.

Тип клиента можно выбрать из пользовательского справочника типов клиентов (если типы были созданы заранее) или создать тип клиента на странице добавления клиента, щелкнув ЛКМ на ссылку «Добавить», находящуюся рядом с полем «Тип».

Созданный тип будет автоматически добавлен в пользовательский справочник.
