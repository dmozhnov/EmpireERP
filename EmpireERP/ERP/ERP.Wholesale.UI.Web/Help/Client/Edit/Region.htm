В данном поле необходимо указать регион клиента.

Регион клиента можно выбрать из пользовательского справочника регионов клиентов (если регионы были созданы заранее) или создать регион клиента на странице добавления клиента, щелкнув ЛКМ на ссылку «Добавить», находящуюся рядом с полем «Регион клиента».

Созданный регион клиента будет автоматически добавлен в пользовательский справочник.
