В данном поле необходимо указать программу обслуживания клиента.

Программу обслуживания клиента можно выбрать из пользовательского справочника программ обслуживания (если программы были созданы заранее) или создать программу обслуживания клиента на странице добавления клиента, щелкнув ЛКМ на ссылку «Добавить», находящуюся рядом с полем «Программа обслуживания».

Созданная программа обслуживания будет автоматически добавлена в пользовательский справочник.
