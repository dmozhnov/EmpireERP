На этой странице представлена следующая информация по выбранному клиенту:

  • Основные детали клиента – отображается информация, введенная при создании (редактировании) клиента, а также важные расчетные показатели по клиенту.
  • Таблица «Связанные задачи и мероприятия» – отображается список всех связанных с данным клиентом мероприятий и задач.
  • Таблица «Сделки» – отображается список всех активных сделок с данным клиентом.
  • Таблица «Реализации по сделкам» – отображается список всех накладных реализации товаров, созданных в рамках любой из сделок с данным клиентом.
  • Таблица «Возвраты от клиента» – отображается список всех накладных возвратов товаров, принятых от клиента в рамках любой из сделок с ним.
  • Таблица «Оплаты по сделкам» – отображается список всех принятых от клиента платежей, сделанных им в рамках любой из сделок с ним.
  • Таблица «Корректировки сальдо по сделкам» – отображается список всех дебетовых и кредитовых корректировок общего сальдо клиента.
  • Таблица «Организации клиента» – отображается список всех связанных с данным клиентом Организаций клиента.
