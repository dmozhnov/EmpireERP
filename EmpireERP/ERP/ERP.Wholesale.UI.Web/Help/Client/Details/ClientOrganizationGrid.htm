В этой таблице выводятся связанные с данным клиентом организации. Договора, созданные (заключенные) с этими организациями можно выбирать для сделок с данным клиентом. И, наоборот, при заключении нового договора в сделке с данным клиентом, любую из этих организаций можно указать одной из сторон договора.

Сразу над таблицей расположена кнопка «Добавить организацию». По нажатию на нее будет предложено добавить одну из существующих организаций данному клиенту или создать новую организацию и добавить ее клиенту.

Кнопка не отображается или недоступна в случаях если:

  • недостаточно прав доступа для выполнения данной операции.
