В данном поле выводится информация о просрочке платежей за сделанные клиенту реализации товаров. В поле отображаются два расчетных параметра, разделенные знаком «||». Первый – максимальный срок просрочки платежа по всем просроченным платежам по любой из сделок. Второй – общая сумма просроченных платежей по всем накладным реализации по всем сделкам.
