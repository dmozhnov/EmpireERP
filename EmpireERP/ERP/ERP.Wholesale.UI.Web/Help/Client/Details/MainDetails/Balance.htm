В данном поле выводится расчетный параметр, отображающий текущее сальдо по всем активным сделкам. Значение может быть как положительным, так и отрицательным. Положительное значение отражает общий долг клиента (суммарно по всем сделкам), отрицательное – Ваш долг перед клиентом (также суммарно по всем сделкам).
