В этой таблице выводятся все накладные реализации товаров, сделанные клиенту в рамках любой из сделок с ним.

Сразу над этой таблицей расположена кнопка «Новая реализация товаров», по нажатию на которой будет предложено создать новую накладную реализации товаров данному клиенту. При создании накладной по этой кнопке необходимо будет указать сделку с данным клиентом.

Кнопка не отображается или недоступна в случаях если:

  • недостаточно прав доступа для создания накладной реализации товаров.

Чтобы перейти в детали накладной реализации товаров, нужно щелкнуть ЛКМ на ссылку этой накладной в столбце «Номер документа».

Данные в полях «Сделка» и «Место хранение» также выводятся ссылками на страницы деталей соответствующих сущностей.

В таблице «Реализации по сделкам» все записи выводятся в порядке, обратном их созданию – последняя созданная накладная реализации товаров выводится первой в таблице.
