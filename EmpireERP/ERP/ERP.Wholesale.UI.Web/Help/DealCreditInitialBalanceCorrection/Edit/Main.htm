Добавление новой кредитовой корректировки сальдо выполняется в два шага. На первом шаге необходимо внести информацию о корректировке сальдо, заполнив данную форму, и нажать кнопку «Начать разнесение» - затем Вы перейдете на второй шаг. На втором шаге будет предложено выполнить разнесение кредитовой корректировки сальдо по накладным реализации или дебетовым корректировкам.
