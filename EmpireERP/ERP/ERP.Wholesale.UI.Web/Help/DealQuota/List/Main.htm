Квота в Bizpulse определяет условия отгрузки товаров для накладных реализации товаров. На данной странице создаются общие корпоративные квоты, т.е. квоты, которые в дальнейшем можно будет добавлять в сделки. Чтобы использовать квоту в накладных реализации товаров, необходимо создать соответствующую общую квоту и добавить эту квоту в ту сделку, в рамках которой будут создаваться накладные реализации товаров.

При создании квоте присваивается статус «Действующая». Это означает, что данную квоту можно добавлять в сделки и использовать в накладных реализации товаров.

Если какая-либо квота становится неактуальной, то необходимо перевести ее статус в «Недействующая». При этом данная квота будет удалена из всех сделок и ее использование во всех новых накладных реализации будет невозможно. В тех накладных, в которых данная квота уже установлена, она будет продолжать действовать.
