Квота определяет следующие ключевые условия реализации товаров:

  • Процент скидки – базовый процент скидки, который будет накладываться на действующую учетную цену товаров в накладной реализации, в результате будет определяться отпускная цена товаров.
  • Условие взаиморасчетов – может быть задано одно из двух значений: предоплата или отсрочка платежа.
  • Срок отсрочки платежа – если выбрана отсрочка платежа, то данным параметром определяется максимальный срок отсрочки платежа в днях до наступления просрочки. Этот срок начинает отсчитываться от даты отгрузки накладной реализации товаров. Если указано значение «0», то отсрочка платежа считается бессрочной по данной квоте.
  • Максимальный кредитный лимит – если выбрана отсрочка платежа, то данным параметром определяется максимальная совокупная стоимость отгруженных и неоплаченных товаров по данной квоте. Считается для накладной, в которой выбрана данная квота, но по всем неоплаченным накладным в сделке.
