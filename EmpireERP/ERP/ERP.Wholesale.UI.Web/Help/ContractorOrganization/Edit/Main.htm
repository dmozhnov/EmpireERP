Для того чтобы в сделках с данным контрагентом можно было выбирать/создавать договор, с контрагентом должна быть связанна хотя бы одна организация.

С помощью данной формы выполняется добавление связи между одной из существующих организаций и контрагентом.

Если необходимо создать новую организацию, то нужно щелкнуть ЛКМ на ссылку «Создать новую организацию...».
