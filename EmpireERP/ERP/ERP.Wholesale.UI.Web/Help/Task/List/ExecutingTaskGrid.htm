В этой таблице выводятся все задачи и мероприятия в статусе «В работе», внесенные в Bizpulse.

Чтобы открыть детали задачи, нужно найти соответствующую строку в таблице и щелкнуть ЛКМ на название этой задачи, представленное в столбце «Тема».

В столбце «Тип» выводится тип задачи из системного справочника.

В столбце «Приоритет» выводится приоритет задачи из системного справочника.

В столбце «% выполнения» выводится процент выполнения задачи (от 0% до 100%).

В столбце «Дата начала исполнения» выводится дата начала исполнения задачи.

В столбце «Требуемое завершение» выводится дата, когда задача должна быть завершена.

В столбце «Автор» выводится пользователь, создавший задачу.

В столбце «Ответственный» выводится пользователь, которому назначена задача.

В таблице «Задачи и мероприятия в работе» все записи выводятся отсортированными по полю «Требуемое завершение» (вверху выводится самое позднее завершение, ниже – самое раннее или неопределенное). Внутри – по полю «Приоритет» (вверху выводится наивысший), затем по полю «Дата начала исполнения».

Оранжевым цветом подсвечиваются задачи, до требуемых дат завершения которых осталось меньше суток.

Красным цветом подсвечиваются задачи, которые уже должны быть завершены, т.е. требуемые даты завершения уже пройдены, а задачи еще в работе.
