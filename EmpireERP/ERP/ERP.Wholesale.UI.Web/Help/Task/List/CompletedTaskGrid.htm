В этой таблице выводятся все задачи и мероприятия в статусах «Выполнена», «Снята», «Завершена», внесенные в Bizpulse.

Чтобы открыть детали задачи, нужно найти соответствующую строку в таблице и щелкнуть ЛКМ на название этой задачи, представленное в столбце «Тема».

В столбце «Тип» выводится тип задачи из системного справочника.

В столбце «Приоритет» выводится приоритет задачи из системного справочника.

В столбце «Дата начала исполнения» выводится дата начала исполнения задачи.

В столбце «Дата завершения» выводится дата выполнения, снятия или завершения задачи.

В столбце «Автор» выводится пользователь, создавший задачу.

В столбце «Ответственный» выводится пользователь, которому назначена задача.

В таблице «Завершенные задачи и мероприятия» все записи выводятся отсортированными по полю «Требуемое завершение» (вверху выводится самое раннее завершение, ниже – самое позднее). Внутри – по полю «Приоритет» (вверху выводится наивысший), затем по полю «Дата начала».

Красным цветом подсвечиваются задачи, завершение которых было просрочено, т.е. задачи завершены после требуемых дат завершения.
