В этой таблице выводятся все должности пользователей, внесенные в Bizpulse.

Сразу над этой таблицей расположена кнопка «Новая должность пользователя» (она отображается и доступна в том случае, если Ваши права доступа позволяют создавать должности пользователей).

Для редактирования должности пользователя нужно нажать ссылку «Ред.» в столбце «Действие».

Для удаления должности пользователя нужно нажать ссылку «Удал.» в столбце «Действие». Удаление возможно только в том случае, если нет ни одного пользователя с данной должностью.
