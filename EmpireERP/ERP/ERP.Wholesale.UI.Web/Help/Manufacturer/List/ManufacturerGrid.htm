В этой таблице выводятся все фабрики-изготовители, внесенные в Bizpulse.

Сразу над этой таблицей расположена кнопка «Новая фабрика-изготовитель» (она отображается и доступна в том случае, если Ваши права доступа позволяют создавать фабрики).

Для редактирования фабрики-изготовителя нужно нажать ссылку «Ред.» в столбце «Действие».

Для удаления фабрики-изготовителя нужно нажать ссылку «Удал.» в столбце «Действие». Удаление фабрики-изготовителя возможно только в случае, если она не задана в качестве параметра ни для одной сущности Bizpulse.
