В этой таблице выводятся все регионы клиентов, внесенные в Bizpulse.

Сразу над этой таблицей расположена кнопка «Новый регион клиента» (она отображается и доступна в том случае, если Ваши права доступа позволяют создавать регионы клиентов).

Для редактирования региона клиента нужно нажать ссылку «Ред.» в столбце «Действие».

Для удаления региона клиента нужно нажать ссылку «Удал.» в столбце «Действие». Удаление региона клиента возможно только в случае, если он не задан в качестве региона клиента ни для одного клиента.
