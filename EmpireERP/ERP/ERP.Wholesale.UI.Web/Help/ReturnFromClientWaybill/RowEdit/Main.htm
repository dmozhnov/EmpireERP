Для добавления позиции в накладную необходимо:

  • выбрать товар,
  • выбрать документ реализации после того, как выбран товар
  • указать количество единиц возвращаемого товара.

Закупочная цена, Отпускная цена, УЦ приемки, Всего реализовано, Возвращено по другим накл., Доступно к возврату будут автоматически определены после выбора документа реализации.
