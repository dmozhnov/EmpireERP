На этой странице представлена информация по выбранной накладной возврата от клиента. Для выполнения ключевых операций над этой накладной служат кнопки и ссылки, расположенные на странице. Такими операциями являются:

  • Добавление позиций в накладную. Операция подразумевает добавление в накладную товара, который ранее был отгружен клиенту. Добавление позиции в накладную возврата возможно только в стадии «Новая накладная». При добавлении позиции требуется указать отгруженные накладные реализации товаров, по которым принимается возврат.

  • Подготовка накладной к проводке. Операция подразумевает перевод накладной в стадию «Готово к проводке» и может быть выполнена только при соблюдении всех следующих условий:

    • стадии накладной соответствует «Новая»,
    • права доступа позволяют выполнить операцию подготовки к проводке для данной накладной.

  • Проводка накладной. Операция подразумевает перевод накладной в стадию «Проведено» и может быть выполнена только при соблюдении всех следующих условий:

    • стадии накладной соответствует «Готово к проводке»,
    • права доступа позволяют выполнить операцию проводки для данной накладной.

  • Приемка накладной. Операция должна быть выполнена, когда возвращаемые от клиента товары поступают на склад. При выполнении операции происходит перевод накладной в стадию «Принято на склад», а в случае, если накладная реализации товаров (по которой принят возврат) оплачена клиентом, то сумма возвращенных товаров освобождается в неразнесенный остаток. Операция приемки может быть выполнена только при соблюдении всех следующих условий:

    • стадии накладной соответствует «Проведено»,
    • права доступа позволяют выполнить операцию приемки данной накладной.

  • Отмена приемки. Операция подразумевает возврат накладной в стадию «Проведено», а при необходимости возврат разнесения оплаты на реализацию (если при приемке накладной возврата товаров такое разнесение было освобождено). Операция может быть выполнена только при соблюдении всех следующих условий:

    • стадии накладной соответствует «Принято на склад»,
    • данная накладная не является источником для любой другой исходящей накладной,
    • права доступа позволяют выполнить операцию отмены отгрузки для данной накладной.

  • Отмена проводки. Операция подразумевает возврат накладной в стадию «Готово к проводке». Операция отмены проводки может быть выполнена только при соблюдении всех следующих условий:

    • стадии накладной соответствует «Проведено»,
    • права доступа позволяют выполнить операцию отмены проводки для данной накладной.

  • Отмена подготовки к проводке. Операция подразумевает возврат накладной в стадию «Новая накладная». Операция может быть выполнена только при соблюдении всех следующих условий:

    • стадии накладной соответствует « Готово к проводке»,
    • права доступа позволяют выполнить операцию отмены подготовки проводки для данной накладной.
  • Удаление накладной (включая все ее позиции). Операция может быть выполнена, только если накладная возврата находится в стадии «Новая накладная» и права доступа позволяют выполнить операцию удаления данной накладной.

  • Редактирование информации о накладной. Операция подразумевает открытие формы редактирования параметров накладной возврата товаров от клиента и может быть выполнена, только если накладная находится в стадии «Новая накладная» и права доступа позволяют выполнить операцию редактирования данной накладной. Для редактирования доступны следующие параметры:

    • Номер накладной
    • Основание для возврата товаров
    • Комментарий
