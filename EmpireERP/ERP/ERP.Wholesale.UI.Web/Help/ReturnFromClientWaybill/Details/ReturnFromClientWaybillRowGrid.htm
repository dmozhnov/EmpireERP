В этой таблице выводятся все текущие позиции созданной накладной возврата товаров от клиента.

Сразу над этой таблицей расположена кнопка «Добавить», по нажатию на которой будет предложено добавить одну позицию накладной. При этом потребуется выбрать наименование принимаемого товара из списка всех реализованных и отгруженных клиенту товаров по указанной в накладной возврата сделке. А также потребуется указать конкретную накладную реализации товаров, по которой принимается данный возврат.

Кнопка не отображается или недоступна в случаях если:

  • недостаточно прав доступа для добавления позиции в накладную возврата от клиента;
  • статус накладной возврата от клиента не позволяет добавлять и редактировать позиции накладной.

В столбце «Действие» для каждой позиции отображаются ссылки:

  • «Ред.» – по нажатию будет открыта форма редактирования позиции накладной,
  • «Удал.» – по нажатию будет выполнено удаление позиции из накладной

или (если статус накладной не позволяет добавлять/редактировать позиции)

  • «Дет.» – по нажатию будет отображена форма детальной информации по данной позиции.

В таблице «Позиции накладной» все записи выводятся единым списком в порядке их добавления в накладную.
