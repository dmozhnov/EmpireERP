В данном поле необходимо указать основание для возврата.

Основание для возврата можно выбрать из пользовательского справочника оснований для возвратов (если основания были созданы заранее) или создать основание для возврата на странице добавления накладной, щелкнув ЛКМ на ссылку «Добавить», находящуюся рядом с полем «Основание».

Созданное основание для возврата будет автоматически добавлено в пользовательский справочник.
