На этой странице представлена следующая информация по выбранной собственной организации:

  • Основные детали организации – отображается информация, введенная при создании (редактировании) собственной организации.
  • Таблица «Счета в российских банках» – отображается список счетов организации в российских банках.
  • Таблица «Счета в иностранных банках» – отображается список счетов организации в иностранных банках.
  • Таблица «Связанные места хранения» - отображается список физических мест, где могут находиться товары, принадлежащие собственной организации.

Чтобы удалить собственную организацию, нужно нажать кнопку «Удалить», находящуюся над основными деталями организации. Удаление невозможно, если организация используется в договорах, накладных и т.д.
