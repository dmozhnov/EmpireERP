В этой таблице выводятся физические места, где могут находиться товары, принадлежащие собственной организации.

Сразу над таблицей расположена кнопка «Добавить». По нажатию на нее будет предложено добавить новое место хранения.

Кнопка не отображается или недоступна в случаях если:

  • недостаточно прав доступа для добавления места хранения.

Для удаления места хранения из списка связанных мест хранения нужно нажать ссылку «Удал. из списка» в столбце «Действие».
