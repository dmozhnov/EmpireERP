Собственная организация – это одна из ключевых справочных сущностей Bizpulse. Собственная организация является собственником товаров, учитываемых в модуле «Управление товарными запасами». Собственная организация является одним из необходимых параметров договоров, учитываемых в Bizpulse, в частности в договорах по сделкам и заказам, а также в договорах с поставщиками.

Собственная организация может быть двух видов:

  • Юридическое лицо.
  • Физическое лицо.
