В этой таблице выводятся все пользователи, которым назначена данная роль.

Сразу над этой таблицей расположена кнопка «Назначить пользователю», по нажатию на которой будет предложено назначить пользователю данную роль.

Кнопка не отображается или недоступна в случаях если:

  • недостаточно прав доступа для назначения роли пользователю.

Правами назначения и лишения роли обладают только пользователи, являющиеся администраторами. Администратор не может ни назначать себе роль, ни лишать себя роли.

Чтобы перейти в детали пользователя, нужно щелкнуть ЛКМ на ссылку этого пользователя в столбце «Пользователь».

Для того чтобы лишить пользователя роли, нужно нажать ссылку «Лишить роли» в столбце «Действие».
