На этой странице представлена следующая информация по выбранной роли пользователя:

  • Основные детали роли пользователя – отображается информация, введенная при создании (редактировании) роли пользователя.
  • Таблица «Роль назначена пользователям» – отображается список всех пользователей, которым назначена данная роль.
  • Таблица «Настройка прав доступа роли» – отображается список всех прав, которыми может обладать пользователь, которому назначена данная роль.

Чтобы редактировать информацию о роли, нужно нажать кнопку «Редактировать», находящуюся над основными деталями роли (она отображается и доступна в том случае, если Ваши права доступа позволяют редактировать информацию о роли).

Удаление роли возможно только в том случае, если ни одному пользователю не назначена данная роль. Чтобы удалить роль, нужно нажать кнопку «Удалить», находящуюся над основными деталями роли. Кнопка «Удалить» видна только тогда, когда роль не назначена ни одному пользователю.
