В Bizpulse для каждого пользователя Вы можете определить набор доступных действий (например, просмотр приходов, отгрузок, изменение принятой оплаты). Но также Вы можете определять индивидуальные наборы данных, над которыми можно выполнять эти действия.

Каждый пользователь может обладать неограниченным набором ролей.
