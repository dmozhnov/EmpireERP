В этой таблице выводятся все типы поставщиков, внесенные в Bizpulse.

Сразу над этой таблицей расположена кнопка «Новый тип поставщика» (она отображается и доступна в том случае, если Ваши права доступа позволяют создавать типы поставщиков).

Для редактирования типа поставщика нужно нажать ссылку «Ред.» в столбце «Действие».

Для удаления типа поставщика нужно нажать ссылку «Удал.» в столбце «Действие». Удаление типа поставщика возможно только в случае, если он не задан в качестве типа поставщика ни для одного поставщика.
