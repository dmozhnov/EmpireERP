На этой странице представлена следующая информация по выбранной организации поставщика:

  • Основные детали организации поставщика – отображается информация, введенная при создании (редактировании) организации поставщика, а также важные расчетные показатели по организации поставщика.
  • Таблица «Закупки у организации» – отображается список всех закупок у данной организации поставщика.
  • Таблица «Договоры с организацией» - отображается список всех договоров, заключенных с данной организацией.
  • Таблица «Счета в российских банках» – отображается список счетов организации поставщика в российских банках.
  • Таблица «Счета в иностранных банках» – отображается список счетов организации поставщика в иностранных банках.

Создать новую организацию на этой странице нельзя. Чтобы добавить организацию поставщика, нужно перейти на страницу «Детали поставщика» и нажать кнопку «Добавить», расположенную над таблицей «Организации».

Чтобы удалить организацию поставщика, нужно нажать кнопку «Удалить», находящуюся над основными деталями организации поставщика. Удаление невозможно, если с организацией поставщика заключен хотя бы один договор.
