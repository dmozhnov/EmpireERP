В этой таблице выводятся пользователи, не имеющие доступа к Вашему аккаунту.

Чтобы открыть детали пользователя, нужно найти соответствующую строку в таблице и щелкнуть ЛКМ на имя пользователя, представленное в столбце «Отображаемое имя».

В столбце «Команды пользователей» выводятся команды, к которым принадлежит данный пользователь.

В столбце «Роли пользователя» выводятся роли, которые назначены данному пользователю.

В столбце «Дата создания» выводится дата создания нового пользователя.

В столбце «Дата блокировки» выводится дата блокировки данного пользователя.

Для того чтобы разблокировать пользователя, т.е. разрешить пользователю доступ к Вашему аккаунту, нужно перейти в детали данного пользователя и нажать ссылку «да» возле пункта «Заблокирован». Это позволит изменить статус пользователя с заблокированного на активный.
