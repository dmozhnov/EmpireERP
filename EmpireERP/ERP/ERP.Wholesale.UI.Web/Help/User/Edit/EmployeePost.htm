В данном поле необходимо указать должность пользователя.

Должность пользователя можно выбрать из пользовательского справочника должностей пользователя (если должности были созданы заранее) или создать должность пользователя на странице добавления должности, щелкнув ЛКМ на ссылку «Добавить», находящуюся рядом с полем «Должность».

Созданная должность будет автоматически добавлена в пользовательский справочник.
