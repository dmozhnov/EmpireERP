В этой таблице выводятся все команды, в которые входит данный пользователь.

Сразу над этой таблицей расположена кнопка «Включить в еще одну команду», по нажатию на которой будет предложено включить пользователя в команду.

Кнопка не отображается или недоступна в случаях если:

  • недостаточно прав доступа для добавления пользователя к команде.

Правами включения в команду и исключения из команды обладают только пользователи, являющиеся администраторами. Администратор не может ни включать сам себя в команду, ни исключать самого себя из команды.

Чтобы перейти в детали команды пользователя, нужно щелкнуть ЛКМ на ссылку этой команды в столбце «Название команды».

Для того чтобы исключить пользователя из команды, нужно нажать ссылку «Исключить из команды» в столбце «Действие».
