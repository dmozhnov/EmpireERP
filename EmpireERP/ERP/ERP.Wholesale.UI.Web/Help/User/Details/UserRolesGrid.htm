В этой таблице выводятся все роли, назначенные данному пользователю.

Сразу над этой таблицей расположена кнопка «Назначить еще одну роль», по нажатию на которой будет предложено назначить роль пользователю.

Кнопка не отображается или недоступна в случаях если:

  • недостаточно прав доступа для назначения роли.

Правами назначения и лишения роли обладают только пользователи, являющиеся администраторами. Администратор не может ни назначать себе роль, ни лишать себя роли.

Чтобы перейти в детали роли пользователя, нужно щелкнуть ЛКМ на ссылку этой роли в столбце «Название роли».

Для того чтобы лишить роли пользователя, нужно нажать ссылку «Лишить роли» в столбце «Действие».
