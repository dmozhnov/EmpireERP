В этой таблице выводятся все задачи и мероприятия, созданные Вами.

Сразу над этой таблицей расположена кнопка «Новая задача» (она отображается и доступна в том случае, если Ваши права доступа позволяют создавать задачи).

Чтобы открыть детали задачи, нужно найти соответствующую строку в таблице и щелкнуть ЛКМ на название этой задачи, представленное в столбце «Тема».

В столбце «Тип» выводится тип задачи из системного справочника.

В столбце «Приоритет» выводится приоритет задачи из системного справочника.

В столбце «Состояние» выводится статус задачи из системного справочника.

В столбце «% выполнения» выводится процент выполнения задачи (от 0% до 100%).

В столбце «Дата начала исполнения» выводится дата начала исполнения задачи.

В столбце «Требуемое завершение» выводится дата, когда задача должна быть завершена.

В столбце «Ответственный» выводится пользователь, которому Вы назначили задачу.

В таблице «Задачи и мероприятия, созданные мной» все записи выводятся отсортированными по полю «Состояние» (вверху выводятся новые задачи, внизу – закрытые). Внутри – по полю «% выполнения» (вверху выводится наименьший процент, внизу - наибольший), затем по полю «Требуемое завершение» (вверху выводится самое позднее завершение, ниже – самое раннее или неопределенное), далее по полю «Приоритет» (вверху выводится наивысший).

Зеленым цветом подсвечиваются задачи, находящиеся в работе, до дат завершения которых осталось больше суток.

Оранжевым цветом подсвечиваются задачи, до требуемых дат завершения которых осталось меньше суток.

Красным цветом подсвечиваются задачи, которые уже должны быть завершены, т.е. требуемые даты завершения уже пройдены, а задачи еще не завершены, или задачи, завершение которых было просрочено, т.е. задачи завершены после требуемых дат завершения.
