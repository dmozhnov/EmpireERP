В этой таблице выводятся все организационно-правовые формы, внесенные в Bizpulse.

Сразу над этой таблицей расположена кнопка «Новая организационно-правовая форма» (она отображается и доступна в том случае, если Ваши права доступа позволяют создавать организационно-правовые формы).

Для редактирования организационно-правовой формы нужно нажать ссылку «Ред.» в столбце «Действие».

Для удаления организационно-правовой формы нужно нажать ссылку «Удал.» в столбце «Действие». Удаление ОПФ возможно только в случае, если она не задана в качестве параметра ни для одной из сущности Bizpulse.
