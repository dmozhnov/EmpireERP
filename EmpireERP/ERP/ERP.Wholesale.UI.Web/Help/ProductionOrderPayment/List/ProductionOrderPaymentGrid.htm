В этой таблице выводятся все оплаты по заказам, регистрирующие сделанные платежи по расходам, связанные со всеми имеющимися заказами на производство.

Оплаты по заказам в Bizpulse можно создавать только со страницы деталей заказа на производство.

В столбце «Заказ» выводится название заказа на производство, для которого сделана данная оплата.

В столбце «Производитель» выводится производитель, выполняющий данный заказ на производство.

В столбце «Номер платежного документа» выводится номер документа, по которому произведена данная оплата.

В столбце «Дата» выводится дата оплаты по заказу.

В столбце «Сумма оплаты в валюте» выводится сумма оплаты по данному заказу в валюте.

В столбце «Валюта» выводится валюта оплаты по заказу.

В столбце «Сумма оплаты в рублях» выводится сумма оплаты по данному заказу в рублях, переконвертированная по текущему курсу валюты.

В столбце «Назначение платежа» выводится назначение платежа оплаты по заказу.

Детальная информация об оплате по заказу открывается в модальной форме по нажатию на ссылку «Дет.» в столбце «Действие».

Для удаления оплаты по заказу нужно нажать ссылку «Удал.» в столбце «Действие».
