В этой таблице выводятся все программы обслуживания клиентов, внесенные в Bizpulse.

Сразу над этой таблицей расположена кнопка «Новая программа обслуживания клиентов» (она отображается и доступна в том случае, если Ваши права доступа позволяют создавать программы обслуживания клиентов).

Для редактирования программы обслуживания клиента нужно нажать ссылку «Ред.» в столбце «Действие».

Для удаления программы обслуживания клиента нужно нажать ссылку «Удал.» в столбце «Действие». Удаление программы обслуживания клиента возможно только в случае, если она не задана в качестве программы обслуживания клиента ни для одного клиента.
