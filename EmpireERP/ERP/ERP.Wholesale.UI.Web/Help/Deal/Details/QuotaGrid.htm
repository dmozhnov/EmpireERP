В этой таблице выводятся квоты, добавленные в данную сделку и доступные к использованию в накладных реализации товаров по данной сделке.

Сразу над таблицей расположены две кнопки «Добавить квоту» и «Добавить все квоты». По нажатию на «Добавить квоту» будет предложено выбрать для добавления в сделку одну из созданных действующих корпоративных квот. По нажатию на «Добавить все квоты» автоматически в сделку будут добавлены все созданные действующие корпоративные квоты.

При удалении квоты из сделки, сама квота не удаляется, а только исключается из использования в накладных реализации по данной сделке.

Квота будет автоматически удалена из данной таблицы при переводе данной корпоративной квоты в состояние недействующей.

В таблице «Квоты по сделке» все записи выводятся в порядке, обратном их добавлению в сделку – последняя созданная запись выводится первой в таблице.
