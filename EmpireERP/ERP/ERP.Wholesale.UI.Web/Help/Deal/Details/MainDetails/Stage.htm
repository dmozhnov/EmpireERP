В данном поле отображается текущий этап сделки из заданного в Bizpulse состава и порядка этапов для сделки. Совокупность всех этапов составляет жизненный цикл сделки. В Bizpulse заданы следующие этапы сделки (в порядке их прохождения):

Активные этапы (сделка в них является активной)

  • Исследование клиента
  • Подготовка коммерческого предложения
  • Согласование / переговоры
  • Подписание договора
  • Исполнение договора
  • Закрытие договора

Закрытые этапы (сделка в них является закрытой)

  • Успешно закрыта
  • Отказ
  • Договор расторгнут
