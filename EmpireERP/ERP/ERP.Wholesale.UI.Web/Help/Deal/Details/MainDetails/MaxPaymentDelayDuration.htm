В данном поле выводится информация о просрочке платежей за сделанные клиенту реализации товаров в рамках данной сделки. В поле отображаются два расчетных параметра, разделенные знаком «||». Первый – максимальный срок просрочки платежа по всем просроченным платежам. Второй – общая сумма просроченных платежей по всем накладным реализации.
