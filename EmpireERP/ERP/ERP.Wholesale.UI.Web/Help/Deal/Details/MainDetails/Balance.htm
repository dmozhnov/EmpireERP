В данном поле выводится расчетный параметр, отображающий текущее сальдо по сделке. Значение может быть как положительным, так и отрицательным. Положительное значение отражает долг клиента по данной сделке, отрицательное – Ваш долг перед клиентом. Значение рассчитывается следующим образом:

Общее сальдо = Сумма реализации – Сумма оплат – Сумма возвратов + Сумма корректировок сальдо
