В данном поле выводится расчетный параметр, отображающий общую сумму товаров по накладным реализации, созданным в рамках данной сделки, ожидающих отгрузку со склада. Товары на эту сумму являются зарезервированными для отгрузки по данной сделке, но по-прежнему числятся в наличии на местах хранения (складах).
