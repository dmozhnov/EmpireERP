В данном поле выводится расчетный параметр, отображающий текущую сумму корректировок сальдо по сделке. Значение показателя может быть как положительным, так и отрицательным. Значение рассчитывается следующим образом:

Сумма корректировок сальдо = Сумма дебетовых корректировок сальдо – Сумма кредитовых корректировок сальдо
