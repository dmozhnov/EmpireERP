В данном поле выводятся реквизиты договора, используемого в сделке. Договор может быть заключен между любой из организаций клиента и любой из собственных организаций. Для сделки можно создать новый договор или использовать ранее созданный (заключенный) договор.

Организации, указанные в договоре по сделке, являются организациями, используемыми в накладных реализации товаров, оплатах и других сущностях сделки.
