На этой странице представлена следующая информация по выбранной сделке:

  • Основные детали сделки – отображается информация, введенная при создании (редактировании) сделки, а также важные расчетные показатели по сделке.
  • Таблица «Связанные задачи и мероприятия» – отображается список всех связанных с данной сделкой мероприятий и задач.
  • Таблица «Реализации по сделке» – отображается список всех накладных реализации товаров, созданных в рамках данной сделки.
  • Таблица «Возвраты по сделке» – отображается список всех накладных возвратов товаров, принятых от клиента в рамках данной сделки.
  • Таблица «Оплаты по сделке» – отображается список всех принятых от клиента платежей, сделанных им в рамках данной сделки.
  • Таблица «Корректировки сальдо по сделке» – отображается список всех дебетовых и кредитовых корректировок общего сальдо по сделке.
  • Таблица «Квоты по сделке» – отображается список доступных по данной сделке квот из числа всех созданных корпоративных квот.
