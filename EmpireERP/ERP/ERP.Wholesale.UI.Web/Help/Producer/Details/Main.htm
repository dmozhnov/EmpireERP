На этой странице представлена детальная информация по выбранному производителю.

Для выполнения ключевых операций, связанных с производителем, служат кнопки и ссылки, представленные на данной странице. К этим операциям относятся:

  • Редактирование информации о производителе. При нажатии на кнопку «Редактировать» будет открыта страница редактирования информации о производителе. Для редактирования доступны следующие параметры:
    • Название производителя
    • Название организации
    • Адрес
    • VAT №
    • Руководитель
    • Ведущий менеджер
    • Контакты
    • Рейтинг
    • Признак, является ли изготовителем
    • Комментарий
  • Удаление производителя. При нажатии на кнопку «Удалить» будет выполнена операция удаления производителя. При этом удаление невозможно в случае, если от этого производителя создан хотя бы один заказ на производство.
  • Добавление счета в российском банке. При добавлении счета необходимо указать его реквизиты, а также банк, в котором открыт данный счет (задается по БИК). Для этого необходимо, чтобы банк был создан в справочнике «Российские банки».
  • Добавление счета в иностранном банке. При добавлении счета в иностранном банке необходимо указать его реквизиты, а также банк, в котором открыт данный счет (задается по SWIFT). Для этого необходимо, чтобы банк был создан в справочнике «Иностранные банки».
  • Создание нового заказа на производство. Создать заказ на производство можно из двух мест: со страницы списка заказов и со страницы деталей производителя. В обоих случаях будет предложена страница ввода информации о заказе. Однако при создании со страницы деталей производителя Вам не потребуется дополнительно выбирать производителя в заказ.
  • Добавление связанной фабрики-изготовителя. Для того чтобы в заказе на производство при вводе производимой продукции указать фабрику-изготовителя этой продукции, данная фабрика должна быть связана с производителем. С производителем может быть связана любая фабрика-изготовитель из соответствующего справочника.
