В этой таблице выводятся связанные с данным производителем фабрики-изготовители. Если производитель сам является изготовителем, то в таблицу автоматически будет добавлена запись о фабрике-изготовителе, соответствующая названию организации данного производителя.

При формировании партии производства продукции в заказе данному производителю в параметре «Фабрика-изготовитель» каждой позиции производства можно будет указать только связанную с производителем фабрику-изготовителя.

Сразу над таблицей расположена кнопка «Добавить». По нажатию на нее будет предложено связать любую из записей в справочнике «Фабрики-изготовители» с данным производителем или создать новую запись и связать ее с производителем.
