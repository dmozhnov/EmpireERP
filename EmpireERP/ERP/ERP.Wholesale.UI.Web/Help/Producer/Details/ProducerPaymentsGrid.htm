В этой таблице выводятся платежи, сделанные данному производителю в рамках любого из заказов на производство. В таблицу попадают только оплаты за производство, так как только этот тип оплат по заказу считается платежом производителю. Детальная информация о платеже открывается в модальной форме по нажатию на ссылку «Дет.» в столбце «Действие». Информация о заказе на производство, по которому был совершен платеж, выводится ссылкой на детали соответствующего заказа.
