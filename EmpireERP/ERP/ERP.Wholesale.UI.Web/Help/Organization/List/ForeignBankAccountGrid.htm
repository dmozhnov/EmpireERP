В этой таблице выводятся счета организации в иностранных банках.

Сразу над таблицей расположена кнопка «Новый счет». По нажатию на нее будет предложено создать новый счет в иностранном банке. При добавлении счета необходимо указать банк (вводится по коду SWIFT), который должен существовать в справочнике «Иностранные банки». Если данного банка нет в справочнике, то его нужно предварительно туда добавить, а после этого создавать счет.

Кнопка не отображается или недоступна в случаях если:

  • недостаточно прав доступа для создания счета организации конрагента в иностранном банке.
