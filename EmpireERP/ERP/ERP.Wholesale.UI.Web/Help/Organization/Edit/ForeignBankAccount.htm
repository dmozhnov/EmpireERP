Данная форма предназначена для добавления/редактирования счета организации в иностранном банке.

При добавлении счета в иностранном банке необходимо указать его реквизиты, а также банк, в котором открыт данный счет (задается по SWIFT). Для этого необходимо, чтобы банк был создан в справочнике «Иностранные банки».
