Данная форма предназначена для добавления/редактирования счета организации в российском банке.

При добавлении счета в российском банке необходимо указать его реквизиты, а также банк, в котором открыт данный счет (задается по БИК). Для этого необходимо, чтобы банк был создан в справочнике «Российские банки».
