В данном поле необходимо указать процент заработной платы продавцу. Если значение «Брать значение % з/п продавцу из группы товара» выставлено в «Нет», то процент необходимо ввести вручную. Если значение «Брать значение % з/п продавцу из группы товара» выставлено в «Да», то процент будет подставлен автоматически из сведений о группе товара.
