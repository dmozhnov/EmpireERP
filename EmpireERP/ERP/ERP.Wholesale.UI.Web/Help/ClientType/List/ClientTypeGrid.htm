В этой таблице выводятся все типы клиентов, внесенные в Bizpulse.

Сразу над этой таблицей расположена кнопка «Новый тип клиента» (она отображается и доступна в том случае, если Ваши права доступа позволяют создавать типы клиентов).

Для редактирования типа клиента нужно нажать ссылку «Ред.» в столбце «Действие».

Для удаления типа клиента нужно нажать ссылку «Удал.» в столбце «Действие». Удаление типа клиента возможно только в случае, если он не задан в качестве типа клиента ни для одного клиента.
