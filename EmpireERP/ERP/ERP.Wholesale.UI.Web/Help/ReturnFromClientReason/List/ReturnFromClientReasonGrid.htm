В этой таблице выводятся все основания для возврата товара от клиента, внесенные в Bizpulse.

Сразу над этой таблицей расположена кнопка «Новое основание для возврата товара от клиента» (она отображается и доступна в том случае, если Ваши права доступа позволяют создавать основания для возврата товаров от клиента).

Для редактирования основания для возврата нужно нажать ссылку «Ред.» в столбце «Действие».

Для удаления основания для возврата нужно нажать ссылку «Удал.» в столбце «Действие». Удаление основания для возврата возможно только в случае, если оно не задано в качестве основания для возврата товаров ни для одной накладной возврата товаров от клиента.
