В данной таблице представлены накладные реализации товаров, на которые может быть разнесена добавляемая оплата (кредитовая корректировка сальдо). В таблицу попадают только проведенные или отгруженные накладные, которые не полностью оплачены (сумма разнесенных на эти накладные платежей меньше чем сумма накладной в отпускных ценах за вычетом суммы возвращенных товаров по данной накладной).
