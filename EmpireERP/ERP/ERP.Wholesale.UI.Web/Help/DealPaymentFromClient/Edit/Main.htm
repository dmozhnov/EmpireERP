Добавление новой оплаты выполняется в два шага. На первом шаге необходимо внести информацию о платеже, заполнив данную форму, и нажать кнопку «Начать разнесение» - затем Вы перейдете на второй шаг. На втором шаге будет предложено выполнить разнесение добавляемой оплаты по накладным реализации или дебетовым корректировкам.
