В этой таблице выводятся все страны, внесенные в Bizpulse.

Сразу над этой таблицей расположена кнопка «Новая страна» (она отображается и доступна в том случае, если Ваши права доступа позволяют создавать страны).

Для редактирования страны нужно нажать ссылку «Ред.» в столбце «Действие».

Для удаления страны нужно нажать ссылку «Удал.» в столбце «Действие». Удаление страны возможно только в случае, если она не задана в качестве параметра ни для одной сущности Bizpulse.
