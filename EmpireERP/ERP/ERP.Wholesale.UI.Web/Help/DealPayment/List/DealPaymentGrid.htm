В этой таблице отображается список всех оплат и возвратов оплат, которые оформлены в Bizpulse и доступны Вам для просмотра. Записи, отражающие возврат оплаты подсвечены желтым цветом, а сумма возврата оплаты выводится со знаком «-».

Сразу над этой таблицей расположены кнопки «Новая оплата по сделке», «Новая оплата по организации» и «Новый возврат оплаты».

По нажатию на кнопку «Новая оплата по сделке» будет открыта форма оформления оплаты по сделке, а затем ее разнесения. В данном случае необходимо будет отдельно указать клиента и сделку с ним, в рамках которой оформляется принятый платеж.

По нажатию на кнопку «Новая оплата по организации» будет открыта форма оформления оплаты по организации, а затем ее разнесения. В данном случае необходимо будет отдельно указать организацию клиента, от которой принят оформляемый платеж.

По нажатию на кнопку «Новый возврат оплаты» будет открыта форма оформления возврата оплаты клиенту. В данном случае необходимо будет отдельно указать клиента и сделку, в рамках которой оформляется возвратный платеж клиенту по ранее сделанной ошибочной оплате.

Кнопки не отображаются или недоступны в случаях если:

  • недостаточно прав доступа для создания оплат или возврата оплат.

В таблице «Оплаты» все записи выводятся в порядке, обратном их дате – последняя по дате оплата выводится первой в таблице.
