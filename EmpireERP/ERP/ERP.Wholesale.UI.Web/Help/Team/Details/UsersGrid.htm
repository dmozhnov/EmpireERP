В этой таблице выводятся все пользователи, включенные в данную команду.

Сразу над этой таблицей расположена кнопка «Добавить пользователя в команду», по нажатию на которой будет предложено добавить пользователя в команду.

Кнопка не отображается или недоступна в случаях если:

  • недостаточно прав доступа для добавления пользователя в команду.

Правами включения в команду и исключения из команды обладают только пользователи, являющиеся администраторами. Администратор не может ни включать сам себя в команду, ни исключать самого себя из команды.

Чтобы перейти в детали пользователя, нужно щелкнуть ЛКМ на ссылку этого пользователя в столбце «Пользователь».

Для того чтобы исключить пользователя из команды, нужно нажать ссылку «Исключить» в столбце «Действие».
