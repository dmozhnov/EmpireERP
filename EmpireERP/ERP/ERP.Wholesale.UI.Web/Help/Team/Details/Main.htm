На этой странице представлена следующая информация по выбранной команде пользователя:

  • Основные детали команды пользователя – отображается информация, введенная при создании (редактировании) команды пользователя.
  • Таблица «Пользователи – члены команды» – отображается список всех пользователей, включенных в данную команду.
  • Таблица «Область видимости – сделки» – отображается список всех сделок, закрепленных за данной командой.
  • Таблица «Область видимости – места хранения» – отображается список всех мест хранения, закрепленных за данной командой.
  • Таблица «Область видимости – заказы на производство» – отображается список всех заказов, закрепленных за данной командой.

Чтобы редактировать информацию о команде, нужно нажать кнопку «Редактировать», находящуюся над основными деталями команды (она отображается и доступна в том случае, если Ваши права доступа позволяют редактировать информацию о команде).

Удаление команды возможно только в том случае, если ни один пользователь не включен в данную команду. Чтобы удалить команду, нужно нажать кнопку «Удалить», находящуюся над основными деталями команды. Кнопка «Удалить» видна только тогда, когда в команду не включен ни один пользователь.
