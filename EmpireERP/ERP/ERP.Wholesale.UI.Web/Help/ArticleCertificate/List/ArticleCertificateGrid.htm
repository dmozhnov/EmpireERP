В этой таблице выводятся все сертификаты товаров, внесенные в Bizpulse.

Сразу над этой таблицей расположена кнопка «Новый сертификат» (она отображается и доступна в том случае, если Ваши права доступа позволяют создавать сертификаты товаров).

Для редактирования сертификата товара нужно нажать ссылку «Ред.» в столбце «Действие».

Для удаления сертификата товара нужно нажать ссылку «Удал.» в столбце «Действие». Удаление сертификата товара возможно только в случае, если он не задан в качестве параметра ни для одной сущности Bizpulse.
