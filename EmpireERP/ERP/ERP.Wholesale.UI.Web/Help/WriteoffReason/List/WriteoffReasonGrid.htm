В этой таблице выводятся все основания для списания, внесенные в Bizpulse.

Сразу над этой таблицей расположена кнопка «Новое основание для списания» (она отображается и доступна в том случае, если Ваши права доступа позволяют создавать основания для списания).

Для редактирования основания для списания нужно нажать ссылку «Ред.» в столбце «Действие».

Для удаления основания для списания нужно нажать ссылку «Удал.» в столбце «Действие». Удаление основания для списания возможно только в случае, если оно не задано в качестве основания для списания ни для одной накладной списания.
