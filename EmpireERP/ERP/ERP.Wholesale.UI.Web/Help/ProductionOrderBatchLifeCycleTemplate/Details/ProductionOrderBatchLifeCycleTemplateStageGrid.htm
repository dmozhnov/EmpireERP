Данная таблица отображает этапы, которые проходит заказ в своем жизненном цикле.

Для добавления этапа жизненного цикла заказа нужно нажать ссылку «Доб.» в столбце «Действие».

Для редактирования этапа нужно нажать ссылку «Ред.» в столбце «Действие».

Для удаления этапа нужно нажать ссылку «Удал.» в столбце «Действие».

Для изменения позиции этапа в жизненном цикле заказа нужно нажать маркер в столбце «Поз.».
