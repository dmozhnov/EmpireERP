На этой странице представлена информация о конкретном шаблоне жизненного цикла заказа.

Создать новый шаблон на этой странице нельзя. Чтобы создать шаблон, нужно перейти на страницу «Шаблоны заказов» и нажать кнопку «Новый шаблон», расположенную над таблицей «Шаблоны жизненного цикла заказа».

Чтобы редактировать шаблон заказа, нужно нажать кнопку «Редактировать», находящуюся над таблицей «Этапы шаблона жизненного цикла заказа».

Чтобы удалить шаблон заказа, нужно нажать кнопку «Удалить», находящуюся над таблицей «Этапы шаблона жизненного цикла заказа».
