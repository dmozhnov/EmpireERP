В этой таблице выводятся организации, которые могут размещать свои товары на данном месте хранения.

Сразу над таблицей расположена кнопка «Добавить». По нажатию на нее будет предложено связать новую организацию с данным местом хранения.

Кнопка не отображается или недоступна в случаях если:

  • недостаточно прав доступа для добавления связанной организации.

Для удаления организации из списка связанных организаций нужно нажать ссылку «Удал. из списка» в столбце «Действие».
