В этой таблице выводятся секции, на которые разделено данное место хранения.

Сразу над таблицей расположена кнопка «Новая секция». По нажатию на нее будет предложено создать новую секцию данного места хранения.

Кнопка не отображается или недоступна в случаях если:

  • недостаточно прав доступа для добавления секции места хранения.

Для редактирования секции места хранения нужно найти соответствующую строку в таблице и щелкнуть ЛКМ на название этой секции места хранения, представленное в столбце «Название».

Для удаления секции места хранения нужно нажать ссылку «Удал. из списка» в столбце «Действие».
