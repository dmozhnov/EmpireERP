На этой странице представлена следующая информация по выбранному месту хранения:

  • Основные детали места хранения – отображается информация, введенная при создании (редактировании) места хранения.
  • Таблица «Связанные организации» – отображается список организаций, за которыми закреплено данное место хранения.
  • Таблица «Секции места хранения» – отображается список секций, на которые разделено место хранения.

Для создания реестра цен нужно нажать кнопку «Создать реестр цен», расположенную над основными деталями места хранения.

Чтобы удалить место хранения, нужно нажать кнопку «Удалить», находящуюся над основными деталями места хранения. Удаление невозможно, если место хранения используется хотя бы в одной накладной.
