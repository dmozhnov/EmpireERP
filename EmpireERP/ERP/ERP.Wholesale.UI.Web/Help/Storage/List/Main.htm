Место хранения в Bizpulse – физическое место, где могут находиться товары. Собственником товаров, хранящихся на местах хранения, считаются организации, к которым прикреплены данные места хранения.

На одном месте хранения могут находиться товары различных организаций. Все эти организации должны быть связаны с данным местом хранения.
