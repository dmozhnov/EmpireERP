В данном поле необходимо указать тип поставщика.

Тип поставщика можно выбрать из пользовательского справочника типов поставщиков (если типы были созданы заранее) или создать тип поставщика на странице добавления поставщика, щелкнув ЛКМ на ссылку «Добавить», находящуюся рядом с полем «Тип».

Созданный тип будет автоматически добавлен в пользовательский справочник.
