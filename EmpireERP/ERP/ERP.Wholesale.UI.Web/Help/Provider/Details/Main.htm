На этой странице представлена детальная информация по конкретному выбранному поставщику. Для выполнения ключевых операций, связанных с поставщиком, служат кнопки и ссылки, представленные на данной странице. К этим операциям относятся:

  • Редактирование информации о поставщике. При нажатии на кнопку «Редактировать» будет открыта страница редактирования информации о поставщике. Для редактирования доступны следующие параметры:
    • Наименование
    • Тип поставщика
    • Надежность
    • Рейтинг
    • Комментарий
  • Удаление поставщика. При нажатии на кнопку «Удалить» будет выполнена операция удаления поставщика. При этом удаление поставщика невозможно в случае, если от этого поставщика внесена хотя бы одна накладная прихода товаров.
  • Добавление организации, связанной с поставщиком. Для ввода накладной прихода товаров необходимо указать договор с поставщиком, а для добавления договора необходимо указать организацию поставщика, с которой заключен этот договор. Таким образом, для работы с поставщиком с ним должна быть связана хотя бы одна организация. С поставщиком можно связать одну из уже существующих организаций или создать новую организацию и связать ее с поставщиком.
  • Добавление договора с поставщиком. Договор с поставщиком указывается в качестве параметра накладной прихода товаров. Таким образом, для оформления закупок у данного поставщика необходимо создать хотя бы один договор с ним.
  • Создание новой накладной прихода товаров от данного поставщика. Накладной прихода товаров оформляется закупка поступающей к учету в Bizpulse продукции. Создать новую накладную можно со страницы деталей поставщика. В этом случае параметр накладной «Поставщик» будет автоматически заполнен.
