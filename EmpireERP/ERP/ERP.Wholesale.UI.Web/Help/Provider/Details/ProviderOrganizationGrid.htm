В этой таблице выводятся связанные с данным поставщиком организации. С поставщиком может быть связана любая существующая организация поставщика, либо может быть создана новая организация и связана с поставщиком. Но создание организации поставщика возможно только со страницы деталей поставщика.

Сразу над таблицей расположена кнопка «Добавить». По нажатию на нее будет предложено добавить одну из существующих организаций данному поставщику или создать новую организацию и добавить ее поставщику.

Кнопка не отображается или недоступна в случаях если:

  • недостаточно прав доступа для выполнения данной операции.
