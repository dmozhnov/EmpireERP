В этой таблице выводятся заключенные с поставщиком договоры, которые можно указывать в качестве параметра в накладных прихода товаров от данного поставщика.

Сразу над таблицей расположена кнопка «Добавить». По нажатию на нее будет предложено создать новый договор, указав все его параметры. Ключевыми параметрами договора являются организации, между которыми заключен данный договор. Указанная собственная организация в договоре будет являться собственником приходуемых товаров накладными, использующими этот договор.

Кнопка не отображается или недоступна в случаях если:

  • недостаточно прав доступа для создания договора с поставщиком.
