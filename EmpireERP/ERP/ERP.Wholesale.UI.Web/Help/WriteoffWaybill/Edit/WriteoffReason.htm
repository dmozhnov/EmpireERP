В данном поле необходимо указать основание для списания.

Основание для списания можно выбрать из пользовательского справочника оснований для списания (если основания были созданы заранее) или создать основание для списания на странице добавления накладной, щелкнув ЛКМ на ссылку «Добавить», находящуюся рядом с полем «Основание».

Созданное основание будет автоматически добавлено в пользовательский справочник.
